# Local Services Booking Frontend

React frontend for the Local Services Booking platform.

## Features

- Public homepage to browse services
- Guest booking (no account required)
- Customer and Provider dashboards
- Service management for providers
- Booking management
- Availability scheduling

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```
VITE_API_URL=https://local-services-booking-backend-production.up.railway.app/api
```

3. Run development server:
```bash
npm run dev
```

## Build

```bash
npm run build
```
